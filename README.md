**Создание платформы по совместному просмотру фильмов с синхронизированным плеером и чатом.**

**Рабочая версия: просмотр фильмов, которые есть на компьютере у админа комнаты**

Администратор комнаты имеет видео на своем компе, грузит его в браузер и, используя WebRTC, раздает остальным.

Для управления логикой комнат, чатом и синхронизацией будет использоваться сервер. Socket.IO применяется для создания канала связи между клиентами и сервером. Через этот канал передаются все команды управления: PLAY, PAUSE, SEEK , а также сообщения чата.
Сервер использует концепцию комнат для изоляции команд и рассылки их только тем пользователям, которые смотрят один и тот же фильм.

Бэкенд: Node.js с фреймворком Express.js и библиотекой Socket.IO 

**Оптимизации: Добавление к платформе каталога фильмов**

1)	Начальная версия: для передачи видеопотока использовать сервер
2)	Продвинутая версия: Использовать WebRTC. Клиент начинает просмотр, загружая небольшой начальный блок видео с сервера. 
   Затем система использует P2P-соединения, чтобы загружать недостающие сегменты видео у других пользователей в комнате, у которых эти сегменты уже есть.
  	Если пир не находит нужный сегмент у других пользователей, то загружает с сервера. Использовать P2P с централизованным сервером, который хранит в базе данных актуальные IP-адреса всех активных пиров, у которых есть файл

**Как использовать:**

Установите зависимости: npm install

Запустите в одном терминале сервер node server.js, в других клиентов - node client.js

**Реализовано на данный момент:**
Консольное приложение с функциями
1) Регистрация по имени
2) Создание комнат
3) Присоединение к текущим комнатам
4) Отправка сообщений в комнату (всем пользователям) и видимость прошлых сообщений при подключении нового пользователя (до 100 сообщений)
5) Синхронизиргованные команды seek, pause и play с помощью SocketIO  для всех пользователей в комнате. Новый пользователь автоматически синхронизируется с текущим состоянием видео
6) Уведомления о присоединении/выходе пользователей и отображение списка пользователей в комнате
   
**Нет видео-плеера в консоли - используется имитация через текстовые сообщения**
